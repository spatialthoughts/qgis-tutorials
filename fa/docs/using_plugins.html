استفاده از پلاگین Plugins¶

هشدار

This tutorial is now obsolete. A new and updated version is available at Using Plugins (QGIS3)

پلاگین ها در QGIS ابزار های مفیدی را به این نرم افزار اضافه می کنند. این پلاگین ها توسط توسعه دهندگان QGIS و سایر کاربران مستقل نوشته شده اند که می خواهند عملکرد اصلی نرم افزار را گسترش دهند ،. این افزونه ها در QGIS در دسترس همه کاربران است.

نمای کلی تمرین¶

در این آموزش با نحوه فعال کردن * پلاگین های داخلی* Core Plugins و همچنین بارگیری و نصب * پلاگین های خارجی * آشنا خواهید شد. همچنین می توانید نحوه قرارگیری پلاگین را از فهرست QGIS پس از نصب بیاموزید.

مراحل¶

پلاگین داخلی Core Plugins¶

پلاگین های داخلی در حال حاضر بخشی از نصب استاندارد QGIS هستند. برای استفاده از این موارد ، فقط باید آنها را فعال کنید.

  1. نرم افزار QGIS بازکنید. بدین منظور به منوی ک Plugins ‣ Manage and Install Plugins.... سپس گزینه زیر منوی Plugin Manager dialog را کلیک نمایید تا پنجره آن باز شود.

  1. حتی اگر این اولین بار است که از QGIS استفاده می کنید ، افزونه های زیادی را در زیر Installed مشاهده می کنید و که در هنگام نصب QGIS نصب شده اند لذا : به این دلیل که آنها * پلاگین های داخلی* هستند.

  1. حال بیایید یکی از افزونه ها را فعال کنیم. در کادر تیک : Spatial Query Plugin علامت بزنید تا این افزونه را فعال کنید. حال شما می توانید از آن استفاده کنید. نکته ای که باید به آن توجه کنید این است که پلاگین ها قادر به تغییر منوهای مختلف هستند و حتی توانایی ایجاد پنل ها و نوار ابزار جدید دارند. گاه پیدان کردن ابزارهای تازه فعال شده مشکل است.. به عنوان راهنمای به جستجو در پلاگین مراجعه کنید. در اینجا توضیحات پلاگین گروه برداری Category: Vector نوشته شده است. این نشان می دهد که این افزونه در زیر منوی Vector 'یافت می شود. حال بر دکمه Close. کلیک کنید.

  1. اکنون که Spatial Query Plugin فعال شده است ، می توانید به قسمت :menuselection:`Vector --> Spatial Query`بروید تا از قابلیت های آن پلاگین استفاده کنید.

پلاگین خارجی Plugins¶

پلاگین های خارجی در مخزن پلاگین ها QGIS Plugins Repository موجود است و قبل از استفاده توسط کاربران باید نصب شوند. یک روش آسان برای مرور و نصب این پلاگین ها استفاده از ابزار: Plugin Manager است.

  1. نرم افزار QGIS بازکنید. بدین منظور به منوی ک Plugins ‣ Manage and Install Plugins.... سپس گزینه زیر منوی Plugin Manager dialog را کلیک نمایید تا پنجره آن باز شود.

  1. بر دکمه تب Get more برای مشاهده فهرست پلاگین نصب شده کلیک نمایید.

  1. برای این تمرین آموزشی ، بیایید افزونه ای به نام 'QuickQKT' را پیدا و نصب کنیم. هنگامی که شروع به تایپ عبارت qui در جعبه search می کنید ، نتایج جستجو را در زیر خواهید دید. روی: QuickWKT کلیک کنید. بعد ،بر روی دکمه :guilabel:`Install جهت "نصب پلاگین" کلیک کنید.

  1. پس از بارگیری و نصب افزونه ، کادر تأیید را مشاهده خواهید کرد.

  1. اگر توجه کرده باشید ، در توضیحات هیچ اشاره ای به دسته پلاگین نشده است. این امر نحوه دسترسی به پلاگین جدید را دشوار می کند. اکثر افزونه های خارجی در زیر فهرست: guilabel: Plugins در QGIS نصب می شوند. روی Plugins ‣ QuickWKT کلیک کنید و پلاگین جدید را مشاهده خواهید کرد. معمولاً افزونه های خارجی نیز یک دکمه در نوار ابزار Plugins نصب می کنند. همچنین می توانید از آن دکمه برای دسترسی به افزونه استفاده کنید.

پلایگن آزمایشی Plugins¶

اکنون شما می دانید که چگونه * پلاگین خارجی * را در QGIS نصب و پیدا کنید. بیایید چند گزینه پیشرفته را بررسی کنیم. گاهی اوقات به دنبال یک افزونه خاص هستید ، اما نمی توانید آن را درتب the Get more بیشتر" پیدا کنید. شاید به این دلیل که پلاگین * آزمایشی * علامت گذاری شده است. در اینجا نحوه نصب افزونه های * آزمایشی * experimental آورده شده است.

  1. open: guilabel: Plugin Manager در منوی : menuselection:` Plugins -> Manage and Install Plugins ... . در تب : guilabel: `Settings کلیک کنید. گزینه ای برای نمایش پلاگین آزمایشی Show also experimental plugins را مشاهده خواهید کرد. برای فعال کردن ، روی کادر تأیید کنار آن کلیک کنید.

  1. تب جدیدی به نام: :guilabel:`New`را مشاهده خواهید کرد. افزونه های آزمایشی تازه فعال شده در اینجا نشان داده می شوند.

توجه

هنگامی که افزونه های آزمایشی را فعال کنید ، تب New به طور موقت ظاهر می شود. دفعه بعدی که :guilabel:`Plugin Manager`باز می کنید: افزونه های آزمایشی در کنار افزونه های معمولی در تب :guilabel:`Get more`نشان داده می شوند.

  1. Let's install a plugin called TimeManager. Click on the plugin name and then Click Install.

  1. اکنون وقتی به پنجره اصلی QGIS باز گردید ، یک صفحه * جدید در پایین پنجره اصلی مشاهده خواهید کرد. این پنل توسط پلاگین TimeManager ایجاد شده است. این یک روش برای افزودن قابلیت های مفید از طریق رابط کاربر پلاگین ها است.

  1. You can enable/disable this panel from View ‣ Panels ‣ Time Manager.
