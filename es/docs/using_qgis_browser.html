Usando QGIS Browser¶

QGIS viene con una aplicación independiente denominada ** QGIS Browser **. Esta es una herramienta auxiliar útil para QGIS y muy útil en la gestión de conjuntos de datos GIS. Los usuarios de ArcGIS podrian pensar que es una aplicación muy similar a ArcCatalog.

Localizando el QGIS Navegador (o explorador de QGIS)¶

Aplicación independiente QGIS Browser¶

QGIS Browser es parte de la aplicación estandard de QGIS

  • Windows: si instalaste QGIS mediante el instalador OSGEO4W, verás QGIS Browser en el menú de inicio de OSGEO4W.

  • Mac: La aplicación está ubicada en QGIS.app/Contents/MacOS/bin/QGIS Browser.app. Puedes crear un symlink a esta aplicación. Navega a la carpeta Aplicación, clic-derecho al ícono QGIS y selecciona Mostrar Contenido del Paquete. Navega a MacOS ‣ bin ‣ QGIS Navegador. Clic-derecho en el ícono QGIS Navegador y selecciona guilabel:Crear Alias. Arrastra el alias QGIS Navegador a la carpeta Aplicaciones. Ahora puedes acceder al QGIS Navegador como cualquier otra aplicación.

  • Linux: Puedes lanzar el navegador QGIS con el comando qbrowser. Está ubicado en el mismo directorio que la aplicación qgis.

Panel Navegador en QGIS¶

Una manera conveniente de acceder al Navegador QGIS es desde dentro de la misma aplicación QGIS Desktop. El panel navegador está ubicado abajo en el panel de la izquierda en QGIS. Clic en la pestaña Navegador para abrir QGIS Navegador. Si no ves la pestaña Navegador, habilítalo haciendo Vista ‣ Paneles ‣ Navegador (Windows y Mac) o Configuración ‣ Paneles ‣ Navegador (Linux).

Procedimiento¶

  1. Ahora exploremos algo de la funcionalidad del Navegador QGIS. Pásate a la aplicación independiente QGIS Navegador. Navega al directorio en tu sistema donde tengas algunos datos SIG. Notarás inmediatamente la ventaja de usar el Navegador. En vez de ver todos los archivos de apoyo y datos no-espaciales, tu ve sólo las capas espaciales que son soportadas por QGIS. Has clic en una capa para seleccionarla.

  1. Cuando seleccionas una capa, podrás ver los Metadatos en la primera pestaña en el panel a mano derecha. Puedes recoger rápidamente información básica del conjunto de datos desde este panel, como el número de elementos, proyección, etc.

  1. Si pasas a la pestaña Previsualizar, tendrás una previsualización del conjunto de datos. Esta es una manera rápida para determinar como luce tu conjunto de datos antes de abrirlo en QGIS.

  1. La última pestaña es la pestaña Atributos. Aquí puedes ver la tabla de atributos del conjunto de datos para tener alguna idea de los campos disponibles y sus valores.

  1. El Navegador QGIS no sólo te da acceso a capas vector e imágenes en tu sistema, pero también a bases de datos y recursos de red. Si usas cualquier dato en-línea mediante WMS, puedes previsualizarlo rápidamente dentro del navegador. Sólo expande las ubicaciones WMS y verás los recursos que has definido. Similarmente, si tienes bases de datos PotsGIS, SpatialLite o MSSQL disponibles, también puedes acceder a ellas.

  1. El Navegador QGIS tiene la habilidad de navegar y abrir archivos zip directamente. Navega hacia cualquier carpeta que contenga archivos zip. Verás que los archivos zip también aparecen como un conjunto de datos apoyado y puedes previsualizarlo como cualquier otro conjunto de datos.

  1. Otra funcionalidad útil es agregar ciertas carpetas en tu sistema como Favoritos. Has clic-derecho sobre cualquier carpeta y elige Agregar como favorito.

Nota

La adición de una carpeta a tu lista de favoritos funciona actualmente sólo desde el panel Navegador en QGIS. Esta funcionalidad no está disponible en la aplicación independiente.

  1. Después de agregar la ubicación como favorito, puede ser fácilmente accedida desde la carpeta Favoritos en el navegador.

  1. Una vez has seleccionado la capa, puedes hacer doble-clic para agregarla al lienzo QGIS. También puedes arrastrar-y-soltar la capa en el lienzo QGIS.

  1. Puedes volver al panel Capas desde la parte de abajo del panel a mano izquierda en QGIS para ver la capa agregada.
