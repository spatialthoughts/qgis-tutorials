Utilizarea Plugin-urilor¶

Atenționare

Acest tutorial este învechit. O versiune actualizată este disponibilă la Using Plugins (QGIS3)

Plugin-urile adaugă caracteristici utile aplicației QGIS. Acestea sunt scrise de către dezvoltatorii QGIS și de către alți utilizatori, independenți, care urmăresc extinderea funcționalității de bază a soft-ului. Plugin-urile sunt puse la dispoziția tuturor utilizatorilor QGIS.

Privire de ansamblu asupra activității¶

În acest tutorial, veți învăța despre activarea Plugin-urilor de bază, și despre descărcarea și instalarea Plugin-urilor externe. Veți înțelege, de asemenea, modul de identificare a unui plugin în meniul QGIS, o dată ce a fost instalat.

Procedura¶

Plugin-uri de bază¶

Plugin-urile de bază sunt deja parte a instalării QGIS standard. Pentru a le utiliza, trebuie doar să le activați.

  1. Deschideți QGIS. Faceți clic pe Plugins ‣ Manage and Install Plugins…, pentru a deschide fereastra de dialog Plugin Manager.

  1. Chiar dacă aceasta este prima dată când folosiți QGIS, veți vedea o mulțime de plugin-uri enumerate în tab-ul Installed. Acest lucru se datorează faptului că ele sunt Plugin-uri de bază, fiind instalate o dată cu aplicația QGIS.

  1. Haideți să activăm unul dintre plugin-uri. Bifați caseta de lângă Spatial Query Plugin. Acest lucru va activa plugin-ul, după care îl veți putea folosiți. Este de remarcat faptul că plugin-urile au capacitatea de a introduce elemente de meniu în diferite locații, pentru a se crea noi panouri și bare de instrumente. Uneori este dificilă găsirea instrumentelor nou activate. Un prim indiciu ar fi să vă uitați la descrierea plugin-ului. În cazul nostru descrierea spune Category: Vector. Aceasta indică faptul că plugin-ul ar putea fi găsit în meniul Vector, o dată ce a fost activat. Clic pe Close.

  1. Atât timp cât Spatial Query Plugin este activat, puteți merge la Vector ‣ Spatial Query pentru a beneficia de funcționalitatea adăugată de către plugin.

Plugin-uri externe¶

Plugin-urile externe sunt disponibile în Depozitul Plugin-urilor QGIS , ele trebuind să fie instalate de către utilizatori, înainte de a fi folosite. O modalitate simplă pentru găsirea și instalarea acestor plugin-uri este de a folosi instrumentul Plugin Manager.

  1. Deschideți QGIS. Faceți clic pe Plugins ‣ Manage and Install Plugins…, pentru a deschide fereastra de dialog Plugin Manager.

  1. Faceți clic pe tab-ul Get more. Veți vedea o listă a plugin-urilor.

  1. Pentru acest tutorial, trebuie să găsiți și să instalați un plugin numit «QuickQKT». Pe măsură ce începeți să tastați qui în câmpul Search, veți observa dedesubt rezultatele căutării. Apăsați pe QuickWKT. Apoi, efectuați clic pe butonul Install plugin, pentru a-l instala.

  1. O dată ce plugin-ul este descărcat și instalat, veți vedea o fereastră de confirmare.

  1. Dacă ați observat, nu a existat nici o menționare a categoriei plugin-ului în descriere. Acest lucru îngreunează determinarea modului în care se poate apela noul plugin. Cele mai multe plugin-uri externe sunt instalate chiar sub meniul Plugins din QGIS. Faceți clic pe Plugins ‣ QuickWKT și veți vedea plugin-ul nou instalat. De obicei, plugin-urile externe instalează și un buton în bara de instrumente Plugins. Folosirea acelui buton reprezintă o altă modalitate de a accesa plugin-ul.

Plugin-uri Experimentale¶

Acum știți cum să instalați și să găsiți un Plugin extern în QGIS. Haideți să explorăm câteva opțiuni avansate. Uneori sunteți în căutarea unui anumit plugin, dar nu-l puteți găsi în fila Get more. Aceasta se poate datora faptului că plugin-ul este marcat Experimental. Iată cum puteți instala plugin-urile experimentale.

  1. Deschideți Plugin Manager din Plugins ‣ Manage and Install Plugins…. Efectuați clic pe tab-ul Settings. Veți vedea o opțiune numită Show also experimental plugins. Faceți clic pe caseta de selectare de lângă ea, pentru a o activa.

  1. Va apărea o nouă filă numită New. Plugin-urile experimentale nou activate vor fi afișate aici.

Notă

Tab-ul New va apărea numai temporar, după ce activați plugin-urile experimentale. Data viitoare când deschideți Plugin Manager, plugin-urile experimentale se vor afișa alături de plugin-urile obișnuite, în tab-ul Get more.

  1. Haideți să instalăm un plugin numit TimeManager. Faceți clic pe numele plugin-ului și apoi pe Install.

  1. Când vă veți întoarce la fereastra principală QGIS, veți vedea un nou Panou în partea de jos a suportului de hartă. Acest panou este creat de către plugin-ul TimeManager. Acesta este un alt mod în care plugin-urile adaugă funcționalități utile în interfața cu utilizatorul.

  1. Puteți activa/dezactiva acest panou din View ‣ Panels ‣ Time Manager.
