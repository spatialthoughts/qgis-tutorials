Analisi dei Punti in un Poligono¶

Nota

In QGIS3, you can perform this analysis using the Join attributes by location (summary) tool in the Processing Toolbox. Use the polygon layer as the Input Layer, the point layer as the Join layer and count as the Summaries to calculate. See Realizzare unioni spaziali (Spatial Joins) (QGIS3) tutorial for step-by-step instructions.
